Project: ASPA

Application for Athletics Events Tracking

About the Project

This project mainly consists of a web application optimized for tablets, intended to be used by athletics judges to fill out field sheets for events, manage competitions, registrations, etc. Additionally, as part of the project, there is also a small mobile application that allows athlete users to register for competitions and check their results.

Technologies used:

  • MySQL
  • Spring Boot
  • Angular
  • Ionic

Web Features

Login and Create Competition

Create Competition Events

Register Athletes and Call Room

Field sheet of an event (for all disciplines except vertical jump)

Field sheet of vertical jump event

Disqualification control

User help panel

Mobile Features

Event registration and results checking
