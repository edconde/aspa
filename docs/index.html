Proyecto: ASPA

Aplicación para el Seguimiento de Pruebas de Atletismo

Sobre el Proyecto

Proyecto que consiste principalmente en una aplicación web optimizada para tabletas, destinada a ser utilizada por jueces de atletismo para cubrir las hojas de campo de las pruebas, gestionar competiciones, inscripciones, etc. Además, como parte del proyecto, también hay una pequeña aplicación móvil que permite a los usuarios atletas inscribirse en competiciones y consultar sus resultados.

Tecnologías utilizadas:

  • MySQL
  • Spring Boot
  • Angular
  • Ionic

Funcionalidades Web

Inicio de sesión y crear competición

Crear pruebas de una competición

Inscribir atletas y cámara llamadas

Hoja de campo de una prueba (para todas las disciplinas excepto salto vertical)

Hoja de campo de prueba de tipo salto vertical

Control de descalificaciones

Panel de ayuda al usuario

Funcionalidades Mobile

Inscripción a pruebas y consulta de resultados
